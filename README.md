# dmarc-finder

This tool fetches dmarc record and flags if they are not correctly set.

## Requirment:


### packages 

- requests
- bcolors
- BeautifulSoup
- sys
- argparse
- os

### python > 3.x 

## usage: 

damrc.py  -d < Valid domain which you want to check for damrc > 


OPTIONS: 

```
-h 	      Shows help options	
-d            Valid domain for which you want to check damrc
  		
```
